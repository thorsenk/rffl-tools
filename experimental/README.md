# Experimental Data Exploration

This directory contains experimental data extraction and exploration scripts.

## Purpose

Explore the complete data available from ESPN API for historical seasons, test new endpoints, and document what data is accessible with authentication.

## Structure

- `{YEAR}-audit/` - Complete data audits for specific seasons
- `scripts/` - Experimental extraction scripts
- `results/` - Analysis and findings

## Notes

- These experiments may use authentication credentials
- Results may vary based on ESPN API availability
- Some endpoints may require different authentication methods
